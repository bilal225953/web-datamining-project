# Data Directory

## Structure

- `raw/` — Raw crawled articles (JSON). Generated by `src/crawl/crawler.py`
- `processed/` — Cleaned text, entities, triples. Generated by `src/ie/extraction.py`
- `kge/` — Train/valid/test splits for KGE. Generated by `src/kge/embeddings.py`
- `reasoning/` — SWRL reasoning outputs. Generated by `src/reason/swrl_reasoning.py`
- `rag/` — RAG evaluation results. Generated by `src/rag/rag_pipeline.py`
- `samples/` — Sample data files for quick testing

## Regenerating Data

Run the full pipeline:
```bash
python run_pipeline.py --step all
```

Or skip crawling and use existing data:
```bash
python run_pipeline.py --step all --skip-crawl
```
